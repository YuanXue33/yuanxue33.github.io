Git学习笔记(持更)

初次使用Git前的配置（用户名和邮箱都是自己的哦）

git config –global user.name “用户名”
git config –global user.email “邮箱”

创建空的Git仓库(在当前文件夹下创建.git文件夹)

git init

将需要进行版本管理的文件放入暂存区域

git add “文件名

将暂存区域的文件提交到Git仓库(commit是提交的意思，-m选项后边跟着的是本次提交的说明)

git commit -m “注释说明”
